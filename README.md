# 🎙️ Халид — Голосовой помощник

Голосовой ассистент на казахском / русском языке. Работает прямо в браузере без серверов.

## 🗂️ Структура файлов

```
khalid-site/
├── index.html   ← главная страница
├── style.css    ← все стили
├── photo.jpg    ← фото ассистента
└── README.md
```

## 🚀 Запуск на GitHub Pages

1. Загрузи все 3 файла (`index.html`, `style.css`, `photo.jpg`) в репозиторий
1. Settings → Pages → Source: **main branch / root**
1. Сайт будет доступен по адресу: `https://ТВО_НИК.github.io/ИМЯ_РЕПО`

## 💬 Команды

- **Привет / Пока** — приветствие
- **Найди [запрос]** — поиск в Яндексе
- **Переведи [слово]** — Google Переводчик
- **Погода [город]** — прогноз
- **Который час / Какое число** — дата и время
- **Пошути** — анекдот
- **YouTube, ВКонтакте, Телеграм…** — открыть сайты
- **Громкость [0-100]** — изменить громкость
- **Команды** — показать все команды

> Лучше работает в Google Chrome
